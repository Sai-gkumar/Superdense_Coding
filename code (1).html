What is Superdense Coding?

Superdense coding is a quantum communication protocol that allows transmission of two classical bits using only one qubit, enabled by quantum entanglement. This demo simulates the complete process from entanglement creation to classical bit recovery.

Key Steps:

  • Create entangled Bell state pair
  • Encode 2 classical bits into qubit state
  • Transmit single qubit to receiver
  • Perform Bell measurement
  • Recover original classical bits

Learning Objectives:

  • Understand quantum entanglement
  • Learn quantum gate operations
  • See quantum measurement in action
  • Grasp quantum information theory

How Superdense Coding Works

Theoretical Foundation

Superdense coding leverages quantum entanglement to achieve what's classically impossible: transmitting two bits using one quantum particle. The key is that entangled particles share correlations that exceed classical limits.

  • Entangled qubits exist in superposition
  • Operations on one qubit affect the other instantly
  • Bell measurement reveals the encoded information
  • No-cloning theorem ensures security
